Felipe's Scheduler

Felipe's Scheduler is a daily planner application for android which allows users to plan around repetitive tasks in their schedule. Tasks are entered by the user, then the app automatically suggests a schedule which fits tasks into specified time slots in the day. Users can then drag tasks into a better time slot until they are satisfied. This application is a powerful productivity and visualization tool. It was developed entirely using Android Studio and makes use of the Joda-Time API and a SQLite database as well as custom Android UI classes. The latest version of the app can be found on the Google Play Store.

feliperonderos.com (this website!)

This portfolio website is deployed on Google App Engine's Python Engine. It features a handy messaging page (check it out at the contact page!). The backend uses Google's Datastore to store message data, Memcache for instant access to messages and the Mail Python API to email me when someone sends a new message (hooray!) . The front end uses the Jinja2 templating engine, JavaScript and Bootstrap version 3. Both use unhashed cookies to keep track of the user's submitted name when a message is sent.
